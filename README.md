# ⌚ REDMI SMART CLOCK PROJECT

Ushbu loyiha **Soat** ekotizimining dizayn falsafasiga asoslangan holda yaratilgan zamonaviy, interaktiv va yuqori aniqlikdagi veb-soat hisoblanadi. Loyiha ham analog, ham raqamli vaqt ko'rsatkichlarini o'zida mujassam etgan.

---

### 📝 LOYIHA HAQIDA
** SMART CLOCK** - bu shunchaki vaqtni ko'rsatuvchi vosita emas, balki foydalanuvchi interfeysi (UI) dizayni va JavaScript algoritmlarining uyg'unligidir.
* **Dizayn:** Xiaomi-ning minimalist "Dark Mode" uslubida ishlangan.
* **Funksionallik:** Real vaqt rejimini millisekundlargacha aniqlikda hisoblaydi.
* **Smart UI:** Batareya, yurak urishi va qadamlar kabi dekorativ "smart-watch" elementlari qo'shilgan.
* **Effektlar:** Shaffoflik (Glassmorphism) va silliq (smooth) animatsiyalar bilan boyitilgan.
